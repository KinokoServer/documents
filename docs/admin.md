# 運営について
当鯖では、一人(Kinoko_2K)のみだと、運営が苦しいためボランティアとして運営を雇っています。

現在の運営数は、8人おり、係数は、5個です。ボランティアなのでお金は発生しません。

現在、運営メンバーの募集を行っております。面接等の可能性があり、メールでの返信になる場合があります。
