# monocraft.netについて
こちらは、外部サービスの「monocraft.net」の登録・ログイン・投票方法について記載されている記事です。

サーバーメンテナンスが時々あり、投票できないことやサーバーページが表示されない場合があります。

そういう場合には、時間をおいて、ページを見ることをお勧めします。また連続でページを更新する行為等は、外部サーバーへの負荷の原因となりかねないのでおやめください。
